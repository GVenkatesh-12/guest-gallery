# Future Feature Roadmap

While the core philosophy of Guest Gallery is simplicity and absolute lockdown, there are several extensibility vectors planned for upcoming releases.

## Phase 1: Core Formats & Media (P0)
- **Secure Video Playback**: Render shared `.mp4` and `.mkv` videos directly in the immersive horizontal pager with minimal control overlays.
- **Secure PDF Viewer**: Support sharing a PDF file to hand over a single document (such as a boarding pass, concert ticket, or slide deck) securely.

## Phase 2: Local Interoperability (P1)
- **Temporary QR Sharing**: Allow generating a local temporary QR code to let guests download a photo directly from your device using local Wi-Fi transfer, without internet storage.
- **Device-to-Device Wireless Push**: Instantly push selected photos to a nearby device running Guest Gallery using local Wi-Fi Direct or Bluetooth LE.

## Phase 3: Wear & Companion Platforms (P2)
- **Wear OS Companion**: Let users quickly authorize exit actions or advance slideshow frames directly from a smartwatch.
- **Android Auto Support**: Support viewing shared location-specific instructions or meeting tickets securely on car dashboard screens.
